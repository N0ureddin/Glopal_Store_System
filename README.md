# Data warehouse
 Data warehouse created using SSIS
 - Subject area: Sales, Customers, Products
 - Fact: oneFact(salesFact)
 - Dimensions: DimCustomer, DimProduct, DimTerritory, DimStatus, DimDate
 - Granuality of Fact: Daily/Product/Customer
